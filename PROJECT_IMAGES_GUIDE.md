## Instructions
1. Download images from the suggested sources
2. Rename them according to the filenames above
3. Save them in the `public/images/` folder
4. Ensure all images are in JPG format
5. Optimize images for web (compress if necessary)

## Color Consistency Tip
Try to choose images with similar color tones or apply a consistent filter to maintain visual cohesion across your projects page.
